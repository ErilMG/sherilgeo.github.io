Final Year Engineering Project

Mitigation of Harmonics Using Sector-Based SVPWM for a 3-Level Inverter-Fed Induction Motor

Final-year engineering project focused on reducing harmonic distortion in induction motor drives using a sector-based Space Vector Pulse Width Modulation (SVPWM) technique for a three-level inverter. The work involved modelling, simulation, and performance evaluation using MATLAB/Simulink.

Comparative analysis was performed between conventional two-level SVPWM, conventional three-level SVPWM, and the proposed sector-based approach using FFT-based Total Harmonic Distortion (THD) analysis, demonstrating improved waveform quality at higher modulation indices.

Tools & Technologies: MATLAB, Simulink, Power Electronics, SVPWM, FFT / THD Analysis
